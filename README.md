# Bugmailer

> A lightweight, free issue tracking and feedback system for freelancers, small startups, and creators.

Bugmailer helps you collect, organize, and manage bugs, feature requests, and user feedback in a simple and efficient way—without the complexity of traditional tools.

---

## Overview

Bugmailer is designed for developers and creators who need a **fast, clean, and easy way** to handle user feedback. It allows anyone to submit issues through a shareable link, making communication between technical and non-technical users seamless.

---

## Features

- **Easy Issue Submission**  
  Submit bugs, feedback, or feature requests with title, description, and optional screenshots.

- **Shareable Project Links**  
  Generate a public link for collecting feedback from clients or users.

- **Clean Dashboard**  
  Manage all issues in a simple and intuitive interface.

- **Status Tracking**  
  Track progress:
  Open → In Progress → Resolved → Closed

- **Real-Time Notifications**  
  Get notified instantly when new issues are submitted.

- **User Roles**  
  Separate permissions for users and project owners/admins.

- **Fast & Minimal**  
  Focused only on essential features—no unnecessary complexity.

---

## Project Goal

Bugmailer focuses on **simplicity, accessibility, and efficiency**.  
Unlike complex tools like Jira or GitHub Issues, it is built specifically for:

- Freelancers  
- Indie developers  
- Small teams  
- Content creators  

---

## Tech Stack

- **Frontend:** React  
- **Backend:** Node.js (Express)  
- **Database:** MongoDB / PostgreSQL  
- **Authentication:** JWT
